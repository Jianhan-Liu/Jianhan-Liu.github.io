Python判断机制

2019-07-07 数据结构与算法 Python

1.Python判空

调用对象的__bool__函数，若不存在该方法，则调用__len__方法，返回0为空，否则为1.如果某类对象没有定义这两种方法，则该类实例化的任何实例判断时均为真。

2.Python判等

2.1 is和==的区别

is根据对象的内存地址判断是否为同一个对象，即调用id()后的地址，如果一致则返回真。

==调用对象的__eq__方法来判断两个对象是否相等，具体返回结果取决于该方法的具体实现。
